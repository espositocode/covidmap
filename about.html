CovidMap.org
Plot covid cases geographically over time.
A project by Jonathan Esposito.


Who is providing the data?
The source data is provided by CovidActNow.

How often is the data updated?
The data is updated daily at midnight UTC with a GitHub Actions workflow.

What is Case Density?
"The number of cases per 100k population calculated using a 7-day rolling average." -CovidActNow

Why is Idaho missing data?
"We have received reports that Idaho is significantly under-reporting cases due to an inability to keep up with the current volume. We don't have accurate case counts to report at this time." -CovidActNow

How was this created?
The frontend is built with D3 and Leaflet, and map tiles are provided by MapBox. The source code is available on GitHub. This website is served entirely with GitHub Pages.
